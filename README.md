# Tisk Task #
O **Tisk Task** é uma aplicação web desenvolvida em Django para o gerenciamento inteligente de tarefas. Diferente de listas de afazeres comuns, o sistema permite a criação de listas independentes para diferentes processos, evitando que o usuário confunda tarefas de setores distintos, como trabalho e afazeres domésticos.
## Funcionalidades Principais: ##
* Criação e Personalização de listas para diferentes contextos.
* Definir a importância das listas (Alta, Média ou Baixa) com feedback visual de cores.
* Barra de progresso automática que indica a porcentagem de tarefas concluídas em cada lista.
* Edição Dinâmica de nomes e prazos das tarefas diretamente na tabela.
* Sistema de Tags para classificar tarefas.
## Tecnologias Utilizadas: ##
* **Backend**: Python 3.x e Django.
* **Frontend**: HTML5, CSS3 e JavaScript.
* **Banco de Dados**: SQLite.
## Estrutura de aplicativos Django: ##
* **Usuários**: Responsável pela autenticação e perfis.
* **Listas**: Gerencia os agrupamentos de tarefas e suas prioridades.
* **Tarefas**: Cuida das entradas individuais, prazos e marcações de conclusão.
